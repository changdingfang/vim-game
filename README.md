
# Vim Game

用于练习vim的操作

![](/imags/vgame.gif)

# 说明

当吃完一个 `hello world` 时即可获得胜利。  

# 操作

| 指令 | 说明 |
| k    | 向上 |
| j    | 向下 |
| h    | 向左 |
| l    | 向右 |
| i    | 开始游戏 |
| 回车 | 结束游戏 |
| f/F  | 向前/向后跳(第二次输入的字符为同一x轴跳跃的位置) | 
| x    | 吃掉角色下面的字符 |

墙体字符: `|`、`-`、`_`、`+`、`=`、`\`、`/`、`{`、`}`、`(`、`)`、`[`、`]`  
跳跃时可穿过的墙体: `+`、`[`、`]`
